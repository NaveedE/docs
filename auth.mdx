# Authentication

Get started with Waves API in under 2 minutes. Authenticate your requests and make your first API call.

## Quick Start

All Waves API requests require an API key passed in the `Authorization` header. Get your API key from the [Waves Dashboard](https://waves.smallest.ai).

## Endpoint

```
POST https://api.smallest.ai/api/v4.0.0/tts/generate
```

## Headers

| Header | Value | Required |
|--------|-------|----------|
| `Authorization` | `Bearer YOUR_API_KEY` | Yes |
| `Content-Type` | `application/json` | Yes |

## Your First API Call

Make a simple text-to-speech request to verify your authentication:

<CodeGroup>

```bash cURL
curl -X POST https://api.smallest.ai/api/v4.0.0/tts/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello from Waves",
    "voice_id": "default"
  }'
```

```python Python
import requests

url = "https://api.smallest.ai/api/v4.0.0/tts/generate"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "text": "Hello from Waves",
    "voice_id": "default"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.smallest.ai/api/v4.0.0/tts/generate', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    text: 'Hello from Waves',
    voice_id: 'default'
  })
});

const data = await response.json();
console.log(data);
```

</CodeGroup>

## Sample Response

```json
{
  "success": true,
  "audio_url": "https://cdn.smallest.ai/audio/abc123.mp3",
  "duration": 1.2,
  "request_id": "req_xyz789"
}
```

## Response Codes

| Code | Description |
|------|-------------|
| `200` | Success - Authentication valid |
| `401` | Unauthorized - Invalid or missing API key |
| `429` | Rate limit exceeded |
